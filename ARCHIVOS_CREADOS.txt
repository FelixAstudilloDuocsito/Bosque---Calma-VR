📁 SCRIPTS DE EDITOR (Assets/Editor/)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Estos scripts añaden opciones al menú "VR Setup" en Unity.

1. VRForestSceneSetup.cs
   ├─ Propósito: Crea automáticamente la escena del bosque VR
   ├─ Qué hace:
   │  • Añade el XR Origin (sistema VR)
   │  • Crea el terreno con texturas
   │  • Distribuye árboles y rocas
   │  • Configura iluminación y niebla
   │  • Añade colisiones a todo
   └─ Menú: VR Setup → Create VR Forest Scene

2. MetaQuestProjectSetup.cs
   ├─ Propósito: Configura Unity para compilar para Meta Quest
   ├─ Qué hace:
   │  • Cambia la plataforma a Android
   │  • Configura IL2CPP y ARM64
   │  • Optimiza configuraciones de calidad
   │  • Ajusta configuraciones gráficas
   └─ Menú: VR Setup → Configure Project for Meta Quest

3. FixMaterialsAndShaders.cs
   ├─ Propósito: Arregla materiales con shaders incorrectos
   ├─ Qué hace:
   │  • Detecta materiales rotos
   │  • Convierte shaders Built-in a URP
   │  • Verifica problemas del proyecto
   └─ Menú: VR Setup → Fix Materials and Shaders
           VR Setup → Check Project Issues

4. QuickSetupMenu.cs
   ├─ Propósito: Wizard que guía paso a paso
   ├─ Qué hace:
   │  • Muestra una ventana con todos los pasos
   │  • Ejecuta cada paso en orden
   │  • Marca los pasos completados
   └─ Menú: VR Setup → 🚀 Quick Setup Wizard

📁 SCRIPTS DE RUNTIME (Assets/Scripts/)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Estos scripts se ejecutan cuando juegas o usas la app en VR.

1. VRLocomotionSetup.cs
   ├─ Propósito: Configura el sistema de movimiento en VR
   ├─ Qué hace:
   │  • Configura velocidad de caminar
   │  • Configura velocidad de girar
   │  • Añade Character Controller para colisiones
   ├─ Se añade a: XR Origin (automáticamente)
   └─ Ajustable: Puedes cambiar velocidades en el Inspector

2. TerrainBoundaryProtection.cs
   ├─ Propósito: Previene que te caigas del terreno
   ├─ Qué hace:
   │  • Mantiene al jugador dentro del área segura
   │  • Detecta si caes bajo altura mínima
   │  • Te respawnea si te caes
   │  • Muestra límites en el editor
   ├─ Se añade a: XR Origin (automáticamente)
   └─ Ajustable: Puedes cambiar área segura y punto de respawn

Si quieres modificar algo:

Cambiar velocidad de movimiento:
   • Abre la escena VRForestScene
   • Selecciona "XR Origin" en la jerarquía
   • En el Inspector, busca "VR Locomotion Setup"
   • Cambia "Move Speed" y "Turn Speed"

Cambiar tamaño del terreno:
   • Edita VRForestSceneSetup.cs
   • Línea 54: terrainData.size = new Vector3(100, 20, 100);
   • Cambia los números (ancho, alto, profundidad)

Añadir más árboles:
   • Edita VRForestSceneSetup.cs
   • Línea 150: for (int i = 0; i < 15; i++)
   • Cambia 15 por el número que quieras

Cambiar área segura:
   • Abre la escena VRForestScene
   • Selecciona "XR Origin"
   • En "Terrain Boundary Protection"
   • Cambia "Safe Area Size"
