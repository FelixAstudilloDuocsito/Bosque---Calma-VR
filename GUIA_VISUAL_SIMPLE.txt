┌───────────────────────────────────────────────────────────────────────────┐
│  OPCIÓN 1: USAR EL WIZARD (MÁS FÁCIL) 🚀                                 │
└───────────────────────────────────────────────────────────────────────────┘

1. En Unity, arriba en el menú:

   VR Setup → 🚀 Quick Setup Wizard

2. Se abrirá una ventana con 4 pasos
3. Haz clic en cada botón en orden
4. ¡Listo!


┌───────────────────────────────────────────────────────────────────────────┐
│  OPCIÓN 2: PASO A PASO MANUAL                                            │
└───────────────────────────────────────────────────────────────────────────┘

╔═══════════════════════════════════════════════════════════════════════════╗
║  PASO 1: VERIFICAR                                                        ║
╚═══════════════════════════════════════════════════════════════════════════╝

   En el menú de Unity:
   
   VR Setup → Check Project Issues
   
   ┌─────────────────┐
   │  VR Setup   ▼   │
   ├─────────────────┤
   │ Check Project..│ ← Haz clic aquí
   │ Fix Materials...│
   │ Configure...    │
   │ Create VR...    │
   └─────────────────┘


╔═══════════════════════════════════════════════════════════════════════════╗
║  PASO 2: ARREGLAR MATERIALES                                              ║
╚═══════════════════════════════════════════════════════════════════════════╝

   En el menú de Unity:
   
   VR Setup → Fix Materials and Shaders
   
   ┌─────────────────┐
   │  VR Setup   ▼   │
   ├─────────────────┤
   │ Check Project...│
   │ Fix Materials..│ ← Haz clic aquí
   │ Configure...    │
   │ Create VR...    │
   └─────────────────┘


╔═══════════════════════════════════════════════════════════════════════════╗
║  PASO 3: CONFIGURAR PARA META QUEST                                       ║
╚═══════════════════════════════════════════════════════════════════════════╝

   En el menú de Unity:
   
   VR Setup → Configure Project for Meta Quest
   
   ┌─────────────────┐
   │  VR Setup   ▼   │
   ├─────────────────┤
   │ Check Project...│
   │ Fix Materials...│
   │ Configure...   │ ← Haz clic aquí
   │ Create VR...    │
   └─────────────────┘


╔═══════════════════════════════════════════════════════════════════════════╗
║  PASO 3.5: HABILITAR OCULUS (IMPORTANTE!)                                 ║
╚═══════════════════════════════════════════════════════════════════════════╝

   Este paso es MANUAL:
   
   1. Edit → Project Settings
   
      ┌─────────────────┐
      │  Edit       ▼   │
      ├─────────────────┤
      │ Preferences...  │
      │ Project Settings│ ← Haz clic aquí
      │ Shortcuts...    │
      └─────────────────┘
   
   2. En la ventana que se abre, busca en el panel izquierdo:
   
      ┌─────────────────────┐
      │ ▼ Player            │
      │ ▼ Quality           │
      │ ▼ XR Plug-in Mgmt  │ ← Haz clic aquí
      │   Graphics          │
      └─────────────────────┘
   
   3. Arriba verás pestañas, haz clic en el ícono de Android:
   
      ┌──────────────────────────────┐
      │ [PC] [Android] [iOS] [...]   │
      │       ↑                       │
      │   Haz clic aquí              │
      └──────────────────────────────┘
   
   4. Marca la casilla "Oculus":
   
      ┌──────────────────────────────┐
      │ Plug-in Providers:           │
      │ ☐ Mock HMD                   │
      │ ☑ Oculus          ← Marca    │
      │ ☐ OpenXR                     │
      └──────────────────────────────┘
   
   5. Cierra la ventana


╔═══════════════════════════════════════════════════════════════════════════╗
║  PASO 4: CREAR LA ESCENA VR                                               ║
╚═══════════════════════════════════════════════════════════════════════════╝

   En el menú de Unity:
   
   VR Setup → Create VR Forest Scene
   
   ┌─────────────────┐
   │  VR Setup   ▼   │
   ├─────────────────┤
   │ Check Project...│
   │ Fix Materials...│
   │ Configure...    │
   │ Create VR...   │ ← Haz clic aquí
   └─────────────────┘
   
   Espera unos segundos... ¡Se creará todo automáticamente!


╔═══════════════════════════════════════════════════════════════════════════╗
║  PASO 5: COMPILAR PARA META QUEST                                         ║
╚═══════════════════════════════════════════════════════════════════════════╝

   1. Conecta tu Meta Quest con el cable USB
   
   2. En los lentes, acepta el permiso de USB debugging
   
   3. En Unity:
   
      File → Build Settings
      
      ┌─────────────────┐
      │  File       ▼   │
      ├─────────────────┤
      │ New Scene       │
      │ Open Scene...   │
      │ Save            │
      │ Build Settings │ ← Haz clic aquí
      └─────────────────┘
   
   4. En la ventana que se abre:
   
      ┌────────────────────────────────┐
      │ Platform:                      │
      │ ☐ PC, Mac & Linux              │
      │ ☑ Android          ← Debe estar│
      │ ☐ iOS                 marcado  │
      │                                │
      │ [Switch Platform] [Build]      │
      │ [Build And Run]    ← Haz clic │
      └────────────────────────────────┘
   
   5. Elige dónde guardar el APK
   
   6. Espera a que compile (puede tardar 10-30 minutos la primera vez)
   
   7. ¡Se instalará automáticamente en tus lentes!

      └─────────────┘              └─────────────┘
